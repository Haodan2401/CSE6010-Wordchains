System: Mac (Xcode)

Path: Using relative path to read the file, please put the coding file and reading file under the same path


Compile: using command 'make' to run Makefile. This will produce 'output'.

Run: Using command './output para1 para2 swag' to run the output. para1 is the source word index, para2 is the destination word index, and swag is optional for graduate requirements. 
Example: ./output 85 100 swag
